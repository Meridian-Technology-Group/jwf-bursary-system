# 03 — Reset a forgotten password

Backlink: [[README#Getting started]]

If you've forgotten your password, you can request a reset link by
email at any time.

## Steps

1. Go to the portal login page.
2. Below the password box, click the **Forgot password?** link.
3. You'll land on a page titled **Reset your password**.
4. Type your email address (the same one the Foundation used to
   invite you) into the **Email address** field.
5. Click **Send reset link**.
6. You'll see a green tick and the message **Check your email**,
   confirming that — if an account exists for that address — a
   reset link has been sent.
7. Open the email from the Foundation, click the reset link, and
   choose a new password (at least 12 characters).
8. After the password is set you'll be returned to the login page
   to sign in.

## If something goes wrong

- **No email arrives.** Wait a couple of minutes, then check your
  spam folder. The Foundation never tells you whether the email
  address you typed is actually registered — that's a privacy
  feature, not a bug. If you genuinely typed the right address and
  still get nothing, contact the bursary team directly.
- **"Too many attempts"** — the system rate-limits reset
  requests. Wait a few minutes and try again.
- **The reset link itself is expired.** Reset links don't stay
  valid forever. Request another one by repeating the steps above.

## Tip

Use a password manager (your phone or browser likely has one
built in) — it removes the need to remember the password at all.
