# 16 — Upload a document

Backlink: [[README#Uploading documents]]

Every required document has its own upload **slot** with a clear
label (e.g. "Birth Certificate", "P60 — Parent 1", "Council Tax
Bill"). The slot lives next to the question that needs it.

## What an empty slot looks like

A dashed box with a cloud icon, the slot label, a short hint
(e.g. "PDF, JPG or PNG"), and a button reading **Choose file**.
You can also **drag and drop** a file from your file manager
straight onto the dashed box.

## Steps

1. Click **Choose file** (or drag-and-drop your file onto the
   box).
2. Pick the file from your device.
3. A progress bar shows the upload running. For most files this
   takes a second or two.
4. The slot turns **green** with a tick, showing:
   - The filename you uploaded.
   - The file size (KB or MB).
   - **View** — opens the file in a new tab so you can confirm
     it's the right one.
   - **Remove** — deletes the upload from the slot.

## Accepted formats

- PDF, JPG / JPEG, or PNG
- Up to **20 MB** per file

See [[19-file-types-and-sizes]] for what to do if a file is
rejected.

## Replacing the file

To upload a corrected version, click **Remove** then upload the
new file the same way. See [[18-replace-a-document]].

## If the upload fails

The slot turns **red** with the message and a **Try again** button.
Common causes:

- **File too large** — over 20 MB. Re-export or compress it.
- **Unsupported file type** — only PDF, JPG, PNG are accepted.
- **Network glitch** — click **Try again**.
