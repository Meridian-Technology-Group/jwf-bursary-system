# 04 — View historical rounds

Backlink: [[README#Round lifecycle]]

Every prior round (any status) remains visible for reporting and
re-assessment comparison.

## Steps

1. Sidebar → **Rounds**. The `/rounds` list shows every round ever
   created, ordered by academic year.
2. Each row displays:
   - **Academic Year**
   - **Status** — DRAFT / OPEN / CLOSED badge
   - **Open Date**, **Close Date**
   - **Pre-Sub** / **Submitted** / **In Progress** / **Complete**
     counts
3. Click the academic year to open `/rounds/[id]`.

## What the round detail page shows

- Round heading, status badge, and the three key dates (Opens,
  Closes, Decision).
- **Summary cards**: Total Applications, Submitted, Assessment In
  Progress, Completed, Qualifies, Does Not Qualify.
- **School breakdown** table: count of applications per school
  (Trinity, Whitgift).
- Action buttons relevant to the round status — for a `CLOSED`
  round these are read-only / disabled.

## Drill-down

- Click into individual applications from a closed round to review
  recommendations, exports, or PDF outputs.
- For cross-round trends, use the **Reports** page (sidebar →
  Reports) which charts data across all rounds.
- For year-on-year comparison of a single bursary holder, open the
  current application — the previous round's data is referenced via
  the assessment's reason codes.
