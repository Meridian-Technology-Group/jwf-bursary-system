# 08 — Section 3: Parent / Guardian Details

Backlink: [[README#Filling in the application]]

This section captures the contact details, relationship status,
and employment of each parent or guardian. If you are a sole
parent, you can tick a box and the Parent 2 fields disappear.

## Before you start

For each parent / guardian:

- Title, full name, date of birth, nationality.
- Home address and postcode.
- Phone number(s) and email.
- Employment status (employed, self-employed, benefits-only,
  pensioner, unemployed) and — if employed — employer name.

## Steps

### Sole parent

1. At the top of the section, answer **Are you a sole parent?**
   - **Yes** — the Parent 2 block is hidden. You only fill in
     Parent 1.
   - **No** — both Parent 1 and Parent 2 blocks are shown.

### For each parent shown

2. **Title** — Mr / Mrs / Ms / Miss / Dr / Prof / Other.
3. **First name** and **Last name** *(required)*.
4. **Relationship status** *(required)* — Single, Married,
   Widowed, Separated, Divorced, In a Civil Partnership, or
   Cohabiting.
5. **Date of birth** *(required)*.
6. **Nationality** *(required)* — pick from the country list.
7. **Home address** — Address line 1, line 2, City/Town, Postcode,
   Country.
8. **Contact details** — phone (mobile and / or home) and email.
9. **Employment status** *(required)* — choose the option that
   best fits:
   - **Employed (PAYE)** — a tax-coded job.
   - **Receiving benefits only (not working)**.
   - **Self-employed — company director**.
   - **Self-employed — sole trader**.
   - **Receiving state / old-age pension**.
   - **Receiving private or occupational pension**.
   - **Unemployed**.
10. Depending on the status, further fields appear (e.g. employer
    name and occupation for PAYE; trading name for self-employed).
    Fill in only what's shown.

### Save

11. Click **Save and Continue**.

## Tips

- The employment status you pick here changes which income rows
  appear in [[12-section-7-parents-income]] — pick carefully.
- If both parents live at the same address, you can copy line by
  line. There's no "same as Parent 1" tickbox here yet.
