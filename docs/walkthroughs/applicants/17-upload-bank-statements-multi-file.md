# 17 — Upload three months of bank statements (multi-file slot)

Backlink: [[README#Uploading documents]]

Some slots accept more than one file. The bank-statement slots in
[[13-section-8-assets-liabilities]] are the main example — you'll
upload three separate monthly statements into one slot per parent.

## How a multi-file slot looks

A dashed drop-zone at the top (same as a normal slot) — but below
it, a **list** of every file you've added so far. Each row in the
list shows the filename, size, **View** link, and a **Remove**
icon.

The drop-zone stays visible the whole time so you can keep adding
more files.

## Steps

1. Click **Choose file** (or drag-and-drop) on the dashed box.
2. Pick the **first** monthly statement.
3. Watch the progress bar. Once it finishes, the file appears as
   a new row in the list below the drop-zone.
4. Click **Choose file** again, pick the **second** statement.
5. Repeat for the **third** statement (and any more if you need
   to provide them).
6. Tip: you can select multiple files at once in the file picker
   and they'll all upload in parallel (up to five at a time).

## How to remove a single file

Click the trash-can icon on its row. The other files stay put.

## How many statements?

Three consecutive months minimum. They should be the three most
recent statements for the parent's main current account. The
account name on the statement should match the parent's name in
section 3.

## Tips

- If you bank with multiple banks, upload statements from each —
  add them all to the same slot.
- Joint-account statements: upload them to **both** parents' slots
  (or upload once and mention in [[14-section-9-additional-information]]).
