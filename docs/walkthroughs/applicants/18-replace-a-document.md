# 18 — Replace a document

Backlink: [[README#Uploading documents]]

Uploaded the wrong file? Or got a clearer copy? You can replace
any document before you submit your application. Once submitted,
the application is read-only.

## Steps

1. Open the section that holds the slot you want to fix.
2. Find the slot — it'll be green with the current filename
   shown.
3. Click **Remove**. The slot reverts to the empty (dashed) state.
4. Click **Choose file** and upload the replacement file.
5. The slot turns green again with the new filename.

For a multi-file slot (e.g. bank statements), you can remove a
specific file with its trash-can icon and add a fresh one without
disturbing the other files in the slot. See
[[17-upload-bank-statements-multi-file]].

## After submission

Once you've clicked **Submit Application**, your uploads are
locked. If the bursary team needs a corrected version, they will
contact you with a "missing documents" request — see
[[24-respond-to-missing-documents-request]].

## Tips

- The system keeps your application's progress markers up to date:
  if a section becomes incomplete because you removed a required
  document, its sidebar indicator turns from green to amber.
- Removing a file deletes it permanently from your application.
  Make sure you have the replacement ready before clicking Remove.
