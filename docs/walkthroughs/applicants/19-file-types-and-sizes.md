# 19 — What file types and sizes are accepted

Backlink: [[README#Uploading documents]]

## Accepted formats

- **PDF** (`.pdf`) — preferred for multi-page documents.
- **JPEG** (`.jpg`, `.jpeg`) — fine for single-page photos.
- **PNG** (`.png`) — fine for screenshots.

## Size limit

**20 MB per file.** That's plenty for a normal PDF or a phone
photo, but very large multi-page scans can exceed it.

## If your file is rejected

The slot will turn red and show one of these messages:

- **"File too large — maximum 20 MB"** — see the tips below.
- **"Unsupported file type — please upload PDF, JPG, or PNG"** —
  convert the file first (see below).

## Reducing file size

- **PDF too big?** Open it in your PDF reader and "Save as
  reduced size" or "Compress PDF". On a Mac, **Preview → File →
  Export → Quartz Filter → Reduce File Size**. On Windows, free
  tools like *PDF24* compress PDFs.
- **Photo too big?** Take a fresh photo from further away, or use
  your phone's "small file" / "medium quality" option.
- **Long scan?** Split the file into two or three smaller PDFs and
  upload them all into the same slot if it's multi-file (e.g.
  bank statements), or contact the bursary team if it's a single-
  file slot.

## Converting to PDF or JPG

- **Word document (.docx)** — open it and "Save as PDF" / "Export
  to PDF".
- **HEIC photo** (iPhone default) — open in Photos, share, and
  pick **JPEG** as the format. Or change your camera setting to
  "Most Compatible" before retaking.
- **Image inside an email** — save the attachment first, then
  upload the saved file.
